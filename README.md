Este projeto faz parte da disciplina Informática Industrial e consiste no desenvolvimento de um sistema de supervisão para automação industrial com Interface Homem-Máquina (IHM).
O sistema realiza monitoramento de variáveis, exibição de alarmes, visualização do estado da linha de produção e comunicação com CLP via OPC UA. Esse projeto foca em desenvolver esse
sistema para linhas de esteiras transportadoras industriais.
